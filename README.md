# Arbitron Trader

Modern trading dashboard built with React + TypeScript + Vite.

## Stack
- React + Vite
- TailwindCSS
- Supabase backend
- Trading UI components

## Setup
```bash
npm install
npm run dev
```

## Environment
Copy `.env.example` to `.env` and add your keys.

## Notes
- Do not commit `.env`
- Rotate any exposed API keys before production deployment.
